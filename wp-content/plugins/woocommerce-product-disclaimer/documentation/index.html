A) Installation - top

  • Upload the plugin to wp-content/plugins.
  • Activate the plugin from the Plugins menu.

B) Features - top

  1. add disclaimer message to woocommerce products
  2. add reject url to disclaimer decline button woocommerce products
  3. the plugin uses an inheritance system to allow you to add a disclaimer to a product only or a category and all its products or all products through general disclaimer
  4. use the same inheritance system on the reject url
  5. you can activate cookies in the same inheritance system so that the user does not have to agree on the same product if he agreed before
  6. settings page to control styling, cookies, disclaimer button labels and general disclaimer to apply on all products
  7. disclaimer message accepts html
  8. the plugin functions on the add to cart in single product and products list (ajax and link)
  9. the plugin functions on both simple and variable products
  10. nice disclaimer ui
  11. responsive view for the disclaimer
